# 用DOCKER搭設CI環境

在/etc/hosts檔中加入

```bash
192.168.1.101 mydomain.dev.com
192.168.1.101 mydomain.dev.gitlab.com
192.168.1.101 mydomain.dev.nexus.com
192.168.1.101 mydomain.dev.redmine.com
192.168.1.101 mydocker.com
```
